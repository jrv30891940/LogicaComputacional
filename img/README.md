# Carpeta de imágenes

Las ilustraciones incluidas (`hero-nodes.svg`, `sistemas-distribuidos.svg`, `reserva-vuelos.svg`,
`ia-sistemas.svg`) son gráficos vectoriales diseñados específicamente para este proyecto, por lo que
no requieren licencia ni atribución.

Para reemplazar cualquiera de ellas por una foto o imagen propia:

1. Guarda tu imagen dentro de esta carpeta (`img/`).
2. Abre `index.html` y busca la etiqueta `<img>` correspondiente.
3. Cambia únicamente el valor de `src="img/nombre-del-archivo"` por el nombre de tu nuevo archivo.

No es necesario modificar el CSS: las imágenes se ajustan automáticamente al tamaño del contenedor.
